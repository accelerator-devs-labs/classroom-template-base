# Estructura Base del Repositorio para GitHub Classroom

Este repositorio define la estructura mínima recomendada para cada assignment de la cohorte.

```text
/
├─ README.md           → Instrucciones de la semana
├─ src/                → Código fuente del estudiante
├─ examples/           → Ejemplos de referencia (opcional)
└─ tests/              → Pruebas automatizadas (opcional)
```

## Reglas generales

- Todo el código del estudiante debe ir dentro de `src/`.
- Los ejemplos oficiales van en `examples/`.
- Las pruebas (si existen) van en `tests/`.
- El archivo `README.md` explica claramente qué hacer, qué se espera y cómo entregar.
